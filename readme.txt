== Installation ==
1. 上传插件压缩包到 /wp-content/plugins/ 文件夹并解压。
2. 在WordPress后台启用插件或直接在WordPress插件后台搜索Typeahead直接在线安装。
3.在您的网站搜索框随便输入试一试，一个美丽搜索下拉框应该会出现在您面前。
4.您还可以去插件设置页面按您的需要设置插件。
